This is (should be) the exact equivalent of the [Cilia code](/comparison/cilia/#by-example) in Cpp2:

- Types
    - `int`, `i32`, `i64`, `f32`
- Variables
    - `n: int = 42;`
    - `n := 42;`
    - `n : const int = 42;`
    - `words: vector<string> = ();`
    - `names: set<string> = ();`
    - `contactInfoForID: map<string, ContactInfo> = ();`
- Functions
    - `multiply: (a: int, b: int) -> int = a * b;`
    - `print: (a: ContactInfo) = { ...; }`
    - `concat: (a: string_view, b: string_view) -> string = ...;`
- Loops
    - `for 1..=10 do (i) { ...; }`
    - `for 0..<words.size() do (i) { ...; }`
    - `for (5, 7, 11, 13) do (i) { ...; }`
    - `for words do (word) { ...; }`

> **Note**  
> In Cpp2 `int` and `42` are typically 32 bits wide, not 64 bits.
