## Comments

- Single-line comments
    - ```
      // if a < b {
      //   TODO
      // }
      ```
- Block comments can be _nested_
    - as in Swift and Rust (unlike C++):
      ```
      /* This
      /* (and this) */
         (and still this)
         is a comment */
      ```
